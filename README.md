# HRIS documentation

## Requirements

* Docker
  * docker-compose
  * docker-machine

## Starting a VM

* source hris.env hris (or hris2)

## Create VM services (initialize)

* docker-compose -f <>.yml create

## Start VM services (initialize)

* docker-compose -f <>.yml start
